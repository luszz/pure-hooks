## ✨ 特性

- 📦 提供一系列实用的纯函数式 React Hooks，方便处理各种任务。
- 🌈 基于 TypeScript 编写，提供详细的类型定义。
- 🎯 轻量、高效，没有不必要的依赖。

## 📦 安装

```bash
npm install pure-hooks
yarn add pure-hooks
pnpm add pure-hooks
```

## 🔨 使用

```ts

import { useToggle } from 'pure-hooks'

```
